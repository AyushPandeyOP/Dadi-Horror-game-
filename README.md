# Dadi: Horror Escape 👻

A browser-based 3D horror escape game built with Three.js.

---

## ▶️ Run Locally

1. Download project
2. Open `index.html` in browser

---

## 🚀 Upload to GitHub

1. Create new repo
2. Upload all files
3. Commit changes

---

## 🌐 Enable GitHub Pages

1. Go to Settings
2. Pages section
3. Select branch: main
4. Save

Your game will be live!

---

## 🎮 Controls

- Joystick → Move
- Run → Faster movement (makes noise)
- Interact → Use objects
- Crouch → Hide
- Drop → Drop items

---

## 🧠 Features

- 3D Horror Environment
- Enemy AI (Dadi)
- Sound-based detection
- Mobile controls
- Ambient horror sound system
